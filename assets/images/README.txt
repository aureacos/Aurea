PUT YOUR PRODUCT PHOTOS HERE
=============================

File naming: use the same name you put in products.js
Example:
  products.js:  images: ['assets/images/moisturiser-01.jpg']
  This folder:  moisturiser-01.jpg

Recommended specs:
  - Format: JPG or WebP (smaller file size)
  - Size: 1200 x 1600px (portrait, 3:4 ratio)
  - Max file size: 500KB per image (use squoosh.app to compress)
  - Background: white, cream, or neutral dark for best results

Multiple images per product:
  images: [
    'assets/images/serum-main.jpg',
    'assets/images/serum-angle.jpg',
    'assets/images/serum-texture.jpg',
  ]
